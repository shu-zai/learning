# 聘错“人”的代价是高昂的
聘错一个人，公司平均要付出15倍于薪水的代价。
## 聘错人的4个原因
1. 不清晰的岗位职责、要求
2. 应聘者寥寥无几
3. 挑不出最合适的人
4. 无法说服看中的人加入
# A级招聘法
本质：一种**招聘理念和方法**  
方式：以**高标准、严要求**来筛选和吸引人才  
目的：招募到最适合岗位需求的**A级选手**（胜任工作、融入企业文化）
## 步骤1：填制记分卡
岗位规划图`（一系列成果和能力的描述）`
1. **使命：工作的实质**  
   用平实的预言写下工作的核心目标
2. **成果：任务必达**
   一个人在岗位上必须干出的业绩
4. **能力：确保胜任**  
   岗位所需的通用能力、专业技能和文化适应性
## 步骤2：物色
在职位空缺前就瞄准新人`（提前为关键岗位，系统化寻觅人才）`
## 步骤3 选拔
四层关卡、层层过滤`（面试收集信息，对照积分卡做决策）`
## 步骤4：说服
“成交”的五大法宝`(确定心仪人选后，说服其加入团队)`
